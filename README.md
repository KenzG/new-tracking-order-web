## Teknologi yang Digunakan

- **Node.js** (server runtime)
- **Express** sebagai web framework

- Vanilla **JavaScript** pada sisi klien (AJAX, event handlers, modal)
- SVG icons & minimal JS untuk interaktivitas
- **dotenv** untuk konfigurasi lingkungan
- **nodemon** untuk pengembangan hot-reload
